### 🗂️ 폴더 구성 규칙
#### ✔ 공통문제
> - 주간 스터디에서 다루는 문제
> - 리뷰 / 코드 비교 목적으로 작성

파일명 규칙:
```java
이름-문제번호.java
```

예)
```java
주정빈-201.java
```

#### ✔ 개인문제
> - 개별 연습 및 추가 문제 풀이 정리
> - 파일명 규칙 동일
